소개

안녕하세요. 저는 Unity와 C#에 특화된 게임 프로그래밍 개발자를 목표로 성장하고 있는 안수민입니다.

Unity 엔진을 활용한 2D 게임 개발과 C# 스크립트를 통한 게임 로직 구현에 관심이 있으며, 캐릭터 이동, 충돌 처리, UI 연동, 게임 진행 흐름 등 실제 플레이에 필요한 기능을 직접 구현하며 개발 역량을 쌓고 있습니다.

단순히 기능을 구현하는 것에서 끝나는 것이 아니라, 유지보수하기 쉽고 확장 가능한 코드 구조를 작성하는 개발자가 되는 것을 목표로 하고 있습니다.

포트폴리오

진주도둑들

진주도둑들은 팀 프로젝트로 개발하여 스토브인디에 출시했던 2D 게임입니다. Unity와 C#을 활용하여 개발을 진행했으며, 실제 게임 출시 과정을 경험할 수 있었습니다.

프로젝트를 통해 C# 기반 게임 로직 구현, 오브젝트 상호작용 처리, UI 구성, 게임 진행 흐름 제작 등 Unity 환경에서의 게임 프로그래밍 경험을 쌓았습니다.
